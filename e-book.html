Introduction to HTML

1.1 Overview

HTML (HyperText Markup Language) is the standard markup language for creating web pages.

1.2 Structure of HTML Document

An HTML document is structured using elements such as tags, attributes, and content.

1.3 Basic HTML Template

                    
                        
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>My HTML Page</title>
                        </head>
                        <body>
                            <h1>Hello, World!</h1>
                        </body>
                        </html>

Heading Tags

2.1 h1 - Heading 1

The h1 tag defines the most important heading on the page.

2.2 h2 - Heading 2

The h2 tag defines a level 2 heading.

2.3 h3 - Heading 3

The h3 tag defines a level 3 heading.

2.4 h4 - Heading 4

The h4 tag defines a level 4 heading.

2.5 h5 - Heading 5

The h5 tag defines a level 5 heading.

2.6 h6 - Heading 6

The h6 tag defines the least important heading.

Attribute Tags

3.1 class Attribute

The class attribute is used to define equal styles for elements with the same class.

                    
                        <p class="important-text">This is an important paragraph.</p>

3.2 id Attribute

The id attribute is used to uniquely identify an element on a page.

                    
                        <div id="header">This is the header</div>

3.3 src Attribute

The src attribute is used to specify the source (URL) of embedded content like images.

                    
                        <img src="image.jpg" alt="An example image">

3.4 alt Attribute

The alt attribute provides alternative text for an image, which is displayed if the image cannot be loaded.

                    
                        <img src="image.jpg" alt="An example image">

3.5 href Attribute

The href attribute is used to specify the URL of a linked resource, typically used with the <a> (anchor) tag.

                    
                        <a href="https://example.com">Visit our website</a>

3.6 style Attribute

The style attribute is used to add inline styles to an element.

                    
                        <p style="color: blue; font-size: 16px;">Styled paragraph</p>

List Tags

4.1 ul - Unordered List

The ul tag is used to create an unordered list.

                    
                        <ul>
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>

4.2 ol - Ordered List

The ol tag is used to create an ordered list.

                    
                        <ol>
                            <li>First item</li>
                            <li>Second item</li>
                            <li>Third item</li>
                        </ol>

4.3 li - List Item

The li tag is used to define a list item.

                    
                        <ul>
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>

4.4 dl - Definition List

The dl tag is used to create a definition list.

                    
                        <dl>
                            <dt>HTML</dt>
                            <dd>HyperText Markup Language</dd>
                            <dt>CSS</dt>
                            <dd>Cascading Style Sheets</dd>
                        </dl>

4.5 dt - Definition Term

The dt tag is used to define a term in a definition list.

                    
                        <dl>
                            <dt>HTML</dt>
                            <dd>HyperText Markup Language</dd>
                            <dt>CSS</dt>
                            <dd>Cascading Style Sheets</dd>
                        </dl>

4.5 dd - Definition Description

The dd tag is used to define the description of a term in a definition list

                    
                        <dl>
                            <dt>HTML</dt>
                            <dd>HyperText Markup Language</dd>
                            <dt>CSS</dt>
                            <dd>Cascading Style Sheets</dd>
                        </dl>
